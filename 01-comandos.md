# Comandos para Laravel

## Basics

Help command
```
$ php artisan help
```

List command
```
$ php artisan list
```

## Servidor
```
# Servidor On 
$ php artisan server
```

## Routes

List Routes
```
$ php artisan route:list
```

## Controller

Create controller
```
$ php artisan make:controller NameController
```

### Resource

```
$ php artisan make:controller NameController --resource
```

## MIDDLEWARE

```
$ php artisan make:middleware NameMiddleware
```

## Database

### Migrate
```
$ php artisan migrate 
```
